Here's everything running under the hood, explained plainly:

The Game App (what you play)

React Native via Expo — this is what makes it a real mobile app that runs on iPhone and Android from a single codebase. It's the same technology used by major apps like Discord and Shopify. You could publish it to the App Store and Google Play.
React Navigation — handles moving between screens (menu → career → match → roster) with smooth transitions
AsyncStorage — saves your career progress locally on the phone, so it persists between sessions like any real game save
The Match Engine

Pure JavaScript/TypeScript logic — the stamina math, cooldown timers, move outcomes, heel action effects, and finisher trigger conditions all live here. The text simulation that exists now is this layer — the visual build wraps around it.
The Visual Layer

React Native's built-in UI components styled with a custom dark/gold design system
Illustrated character art (generated from your photos as style references) displayed as image assets
Animated transitions for move impacts, stamina bar drains, and win/loss sequences using React Native Animated
The Content Layer

All career chapter data, promo dialogue, roster bios, and match records stored as structured data files (JSON/TypeScript objects) — pulled directly from the Master Archive
No internet connection needed to play — everything runs offline on the device
The API Server

A lightweight Express.js backend already running in the project — available if you ever want leaderboards, cloud saves, or future online features, but the game works fully without it for now
